# Frontend Practice Repository

This repository contains front‑end design examples and assignments for the Web Engineering and Development (WED) 201 course. Each folder demonstrates a different project or exercise.

## Contents
- **Apple** – product landing page
- **De ban cam co js** – table example
- **De laptop** – laptop shop layout
- **De xe co js menu** – motorcycle website
- **Mui Ne** – travel site layout
- **Resource_FA24** – shared assets
- ...and others
